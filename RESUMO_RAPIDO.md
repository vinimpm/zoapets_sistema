## 🚀 Como Rodar

### 1. Infraestrutura
```bash
docker-compose up -d
# Aguarde 30 segundos
```

### 2. Backend
```bash
cd backend
npm install
npm run start:dev
# Roda em http://localhost:3000
```

### 3. Frontend
```bash
cd frontend
npm install
npm run dev
# Roda em http://localhost:3001
```

### 4. Login
